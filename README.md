## English

Welcome to **Rootimes's Code ICU**, a personal blog and knowledge base dedicated to software engineering, algorithms, and technical insights.

- **Live Site:** [https://rootimes.github.io/my-logs/](https://rootimes.github.io/my-logs/)
- **Author:** Rootimes ([rootimes8596@gmail.com](mailto:rootimes8596@gmail.com))

### 🚀 Features

- **Multi-language Support:** Traditional Chinese (default) and English.
- **Theme:** Powered by [hugo-theme-tailwind](https://github.com/tomowang/hugo-theme-tailwind), a clean and responsive Tailwind CSS theme.
- **Dark Mode:** Built-in support for light/dark themes.
- **Search:** Fully functional local search.
- **Comments:** Integrated with [Giscus](https://giscus.app/) (GitHub Discussions).
- **SEO Optimized:** JSON-LD, sitemap, and meta tags.
